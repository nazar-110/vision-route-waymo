# Demo Assets

This folder contains a small demo extract generated from a Waymo Open Dataset Perception segment for research illustration.

Included files:

- `prediction_overlay.mp4`: short prediction-only route overlay clip
- `prediction_overlay_first_frame.png`: first-frame camera overlay preview
- `bev_comparison.png`: bird's-eye-view trajectory plot
- `sequence_metrics.json`: metrics for the short rendered sequence

Raw Waymo TFRecords, model checkpoints, and full generated output directories are intentionally not committed.
